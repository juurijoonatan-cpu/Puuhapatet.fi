# Puuhapatet.fi — Claude Code ohjeet

## Pull requestit
Luo aina GitHub PR kun muutokset on commitattu ja pushattu branchille.
PR-kuvaus suomeksi: mitä muutettiin ja miksi.

## Kehityshaara
- Kehitä aina omalla feature-branchilla (älä pushaa suoraan mainiin)
- Branch-nimeäminen: `claude/<lyhyt-kuvaus>-<id>`
